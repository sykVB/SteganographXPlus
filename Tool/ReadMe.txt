SteganographX Plus a freeware by LeeLu Soft 2010 v2.0
-----------------------------------------------------------------
SteganographX Plus is a  security / graphic tool that have two main functions:

1. hide (encrypt) a text in a bitmap image, the text will not be visible and the changes to the bitmap image will not be noticeable in BMP's with 16, 24 or 32-bit colors.
2. Encrypt a bitmap image

You can use each one of the functions separately or use both of them together.

How to use SteganographX 
--------------------------------
SteganographX is a portable application, There is no install process, just unzip the files to any folder and run it.

Using SteganographX like all other LeeLu Soft programs, is very simple.

Hide a text in a bitmap
-------------------------
To hide a text inside a bitmap image you need to provide only two inputs:
Load a bitmap file - This can be done in several ways:
- Go to File manu > Open..., and select the bitmap image to load
- Click on the Open... Button on the side bar and select the bitmap image to load
- Click on the black image area and select the bitmap image to load
- Right click on the black image area and from the popup menu select Open...
Note: after loading the bitmap watch for the status bar information and the maximum characters that you can hide in this image

Enter the text to hide:
- Type the text to hide directly into the text box or
- Paste the text to hide from the clipboard into the text box

Click Insert text, this will insert the text into the bitmap.
Now you can save the image with the hidden text, click on Save...

To Extract a hidden text from a bitmap image:
Load a bitmap with hidden text that was created with SteganographX or SteganographX Plus, this can be done in several ways:
- Go to File manu > Open..., and select the bitmap image to load
- Click on the Open Button on the side bar and select the bitmap image to load
- Click on the black image area and select the bitmap image to load

Click Extract Hidden Text, this will extract the hidden text into the text box.

If you are not sure that SteganographX is secure enough, or for any other reason, you can use SteganographX Plus to encrypt the image.

Encrypt a bitmap image
---------------------------
Open and load a bitmap image (you can do it on bitmap with hidden text or with any other bitmap image)
Enter the encryption key, it must be numbers only, this is the encryption / decryption key, so you must remember this number.
Click the Encrypt / Decrypt once, now you will see that the image is scrambled.
To save the encrypted image, click on Save...

Decrypt a bitmap image
---------------------------
Open and load a bitmap that was encrypted with SteganographX Plus.
Enter the exact encryption key and click Encrypt / Decrypt once.
You can now save the decrypted image


Other features:
Hide / Show side bar 
Clear - to clear the text box and close the image
About - display about SteganographX
Exit - to exit the application


OS Support
--------------
SteganographX Plus was tested on Windows XP Pro SP3 and Windows 7 Ultimate (32bit).
